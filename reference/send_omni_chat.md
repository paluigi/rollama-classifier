# Send chat completion request to OpenAI-compatible server

Send chat completion request to OpenAI-compatible server

## Usage

``` r
send_omni_chat(base_url, api_key, timeout, body)
```

## Arguments

- base_url:

  Character. Base URL.

- api_key:

  Character. API key.

- timeout:

  Numeric. Timeout in seconds.

- body:

  List. Request body.

## Value

Parsed response list.
