# Scoring utilities for classification

Probability and scoring functions used by the adaptive
[`generate()`](https://paluigi-moltis.github.io/rollama-classifier/reference/generate.md)
and multi-call
[`classify()`](https://paluigi-moltis.github.io/rollama-classifier/reference/classify.md)
methods. All length normalization uses geometric mean, applied
consistently across both methods.
